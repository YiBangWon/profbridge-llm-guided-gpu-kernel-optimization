# Limitations

- This is a research prototype, not a production optimizer.
- Profiling reduction alone is not a sufficient compiler-paper contribution.
- Feedback-error margins in the curated run were tiny.
- Uncertainty-only was a strong baseline.
- ProfileSketch was more useful as an acquisition/control representation than as a universally better prompt representation.
- The repository does not claim to beat existing LLM compiler optimization systems.
