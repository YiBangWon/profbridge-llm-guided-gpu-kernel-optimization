# Lessons Learned

This project clarified which claims are defensible and which require stronger performance-seeking evidence.

Useful engineering outcomes:

- GPU profiling and metric parsing infrastructure.
- Candidate correctness, timing, and profiling automation.
- ProfileSketch and Value-of-Profile prototypes.
- Statistical sanity checks and claim recalibration.

Research lesson: a compiler paper needs a performance-seeking story, not only overhead reduction.
